# **App Name**: منوی دیجیتال کارن کافه

## Core Features:

- نمایش دسته بندی‌ها: نمایش دسته‌بندی‌های منو (نوشیدنی، غذا، دسر و غیره) با قابلیت فیلتر کردن.
- نمایش آیتم‌ها: نمایش نام، قیمت و آیکون اختصاصی برای هر آیتم.
- تعامل با کارت‌ها: ایجاد افکت‌های تعاملی کوچک هنگام قرار گرفتن نشانگر موس روی کارت‌ها.

## Style Guidelines:

- رنگ اصلی: قهوه‌ای روشن (#A67B5B) برای حس گرما و صمیمیت.
- رنگ پس‌زمینه: کرم (#F5F5DC) برای ایجاد ظاهری ملایم و آرام‌بخش.
- رنگ تأکید: طلایی نرم (#D4A373) برای برجسته کردن عناصر مهم.
- فونت بدنه و عنوان: 'PT Sans' برای خوانایی بالا و حس مدرن.
- استفاده از آیکون‌های سفارشی مرتبط با هر دسته‌بندی و آیتم.
- طراحی مینیمالیستی و راست‌چین برای راحتی و زیبایی بصری.
- انیمیشن‌های کوچک هنگام تعامل با آیتم‌ها برای جذابیت بیشتر.
